# ValidadorDNI_NIE
Pequeña aplicación en Java para validar DNI o NIE. Con GUI en Swing. Para estudiantes.

El proyecto tiene varias clases: 

Una clase VerifyNIF - que verificará un NIF en formato DNI - ocho letras y un número // ej - "99999999X"

Una clase VerifyNIE - que verificará un NIE en formato "letra - siete numeros - letra" // ej - "0XXXXXXX0"

Una clase main.

Una clase MainFrame que contiene la GUI.

Una clase controladora que comunica los elementos de la GUI con la clase verificadora que corresponda.
